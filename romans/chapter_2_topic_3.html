耶稣基督的使徒保罗

第2章: 保罗与福音 · 主题3

保罗是仆人，但他不是一般的仆人，他以特殊的方式服侍神。他以什么方式服侍神呢？他是一个使徒。使徒是什么意思？使徒就是大使的意思。他们见过复活的基督，换句话讲，他们是“第一手的见证人”。他们不是“道听途说”，不是听别人讲，他们是亲眼见过的。大家今天在读圣经的时候，要知道保罗、彼得、约翰，他们都是第一手的见证人，他们都是亲自从主耶稣基督那里领受。就是说，这位复活的主、这位创造天地万物的神，降世为人，来到这世间，祂的名字叫耶稣，曾经显现给约翰、彼得、保罗。他们受了福音的教导和启示，他们被神赋予特别的权柄和职分。圣经所讲的职分包括使徒、先知、传福音的、牧师、教师，你有没有发现使徒是排第一位的？在基督赐给教会的所有职分当中，使徒是超越其他一切职分之上的。所以，我们现在所读的这些使徒的书信是非常重要的。当时的教会是听使徒的教导才能保持信仰上的纯正。这些使徒的教导能够使整个教会清楚到底我们所信的福音是什么。而且借着使徒的书信，他们的生命不断经历成长。保罗也强调，这种职分不是自封的，不是某个人自认为“哇！我是使徒啊！我看保罗挺威风的，我很羡慕他的这个职分，我也想要做一个使徒。 ”不是这样的。使徒是神特别的拣选，直接领受耶稣复活之后的启示。接下来，我给大家看几段经文。

并且要以我主长久忍耐为得救的因由，就如我们所亲爱的兄弟保罗，照着所赐给他的智慧写了信给你们。他一切的信上也都是讲论这事。信中有些难明白的，那无学问、不坚固的人强解，如强解别的经书一样，就自取沉沦。 (彼得后书 3:15-16)

你看，彼得没有把保罗的书信当做一封普通的书信。这里讲的经书就是圣经的经文。当圣经的正典还没有形成的时候，保罗所写的书信在在彼得的心里已经与圣经的其他经卷相提并论了。这是一个值得思考的情况：彼得认为保罗的书信不能强解，而是需要神格外的帮助才能明白的。

接下来，我们来看一下保罗是如何描述他自己的。

我当日所领受又传给你们的：第一，就是基督照圣经所说，为我们的罪死了，而且埋葬了；又照圣经所说，第三天复活了，并且显给矶法看，然后显给十二使徒看；后来一时显给五百多弟兄看，其中一大半到如今还在，却也有已经睡了的。以后显给雅各看，再显给众使徒看，末了也显给我看； (哥林多前书 15:3-11)

复活的主，不是在复活之后悄悄藏起来，他显给十二使徒看，他显给五百多弟兄看。大家知道基督教会是怎么建立起来的？是通过这些第一手的见证人得以建立的。他们当时在讲道的信息是很简单的：我见过主，他是真的，他复活了！你要信！他们不需要很多学问，他们只要把他们所看见的说出来。然后保罗接下来怎么说呢？他说“显给雅各看，再显给众使徒看，末了也显给我看” 我在这里要强调什么？保罗他到底是不是使徒？他又不是十二使徒之一，他也没有像彼得、约翰、雅各一样跟随耶稣那么多年。但是圣经明确记载，这位保罗见过复活的主，他亲眼见过复活的主。他不是做梦梦到，他是亲眼见过复活的主，否则在这里他也不会说“他显给我看”。

在《使徒行传》中记载保罗的经历：

我说，主阿，你是谁。主说，我就是你所逼迫的耶稣。你起来站着，我特意向你显现，要派你作执事作见证，将你所看见的事，和我将要指示你的事，证明出来。(使徒行传 26:15-16)

所以你看，耶稣对他说“我特意向你显现”，你要看见我。为什么呢？因为接下来你有一个特别的使命，你看见的事，我指示给你的事，你要表明出来，你要传扬出去。

我也要救你脱离百姓和外邦人的手。我差你到他们那里去，要叫他们的眼睛得开，从黑暗中归向光明，从撒但权下归向神；又因信我，得蒙赦罪，和一切成圣的人同得基业。(使徒行传 26:15-18)

这是保罗使命的一个归纳。他要被差派到外邦人中，他去的很多地方是从来没有人听过耶和华的，是从来不知道有一位上帝的。但是神却要使他去传这福音，使他们的眼睛得开，使他们从黑暗归向光明。神赐给他这个使命，神给他什么很特别的东西呢？神向他显现。耶稣说： “我特意向你显现”。

当《圣经》里面谈到使徒，这是非常重要的职分，谁都不要拍着胸脯宣告自己是一个使徒。在《圣经》中所提到的使徒，是当年的十二使徒，包括保罗这位使徒，他们是亲眼见过主的。我们可能是在梦中，异象当中，读经文当中，圣灵在我们心里作工，使我们在信心里面看见主的。他们是用肉眼看见主的，他们是第一手的见证人。

接下来，我们一起读《加拉太书》第 1 章 11-12 节。

弟兄们，我告诉你们，我素来所传的福音不是出于人的意思。因为我不是从人领受的，也不是人教导我的，乃是从耶稣基督启示来的。 (加拉太书 1:11-12)

使徒的另外一层意思是，他们直接从神那里领受。当时的教会是听使徒的教训。那使徒听谁的教训？是神亲自教训他们的，直接启示他们的。保罗之所以是使徒，他说不是人教导我的，不是我去查考哪一本书学到的，是耶稣基督直接启示我的。

也有人引用这段经文，他说： “你看，我们只听神的声音，不听人的声音。 ”这是保罗在说他自己，有些弟兄姊妹把它用在自己的身上，他说： “你看，我就像保罗一样，我也不从人那里领受” 。作为牧师，我们的责任不是从神那里得到启示，我们的责任是解释圣经而已。从两年前我们开始晨祷，我就鼓励大家，我们要的查经不是死气沉沉的查经，或者拿了一大堆解经书、资料，书山书海，在那边讲这卷书的背景是什么。当然我们要有所了解，但不是无止境地在那边谈论一些知识。我说，我们需要的查经是什么样的查经？是圣灵充满的查经。神啊，求你开启我，让我明白你的话语。我要告诉大家，不要超出圣经以外去寻找启示，这是非常危险的。但是你也不要只是说读经，却没有圣灵的带领。你要恳求圣灵。圣灵啊，在我读这段经文的时候帮助我。在你读完后把圣经先放在一边，开始默祷、默想，求神开启你。你要的查经是圣灵充满的。学习神的话语，不是用你的头脑，而是你的灵吃着神给你的东西。

作为牧师，当我们讲道的时候，不是对着人的头脑讲话。当然有一部分是真理的知识，是对着头脑讲话的，但是更重要的是，借着这些经文，借着圣灵的开启，我们在对人的灵魂说话。神所要借着圣经成就的，就是对我们的灵讲话，开启我们的灵。我提到《加拉太书》的这个例子，是要向大家说明什么是使徒。为什么我们今天有可靠的教导，为什么我们今天读保罗的书信呢？他写的信即使很好，但是至于成为经文吗？至于成为神的话吗？这不是保罗的话吗？这是圣灵默示他的，这是圣灵启示他的。那今天还有没有这样的启示呢？没有了。今天我们所有的启示，都不能超出圣经的范围。当然，圣灵可以给我们一些个人的，关于《圣经》启示的一些应用。 《圣经》的启示是有非常多的应用，这些应用在每个人身上是不一样的。我现在所做的是 interpretation，就是“解释”，但是不要忘了， 《圣经》的另外一个层面是 application，就是“应用”。我的责任是“解释”，至于“应用”的话，我会讲一部分。但是同一段经文，每个人的

应用有可能不一样。为什么？因为圣灵才是那最终的老师，在教导着每一个人。所以我作为传道人，要按照正义分解真理的道，把这道解释出来，最后的应用很多时候是圣灵的工作。

接下来有两段经文，第一段是《罗马书》 11 章 13 节。

我对你们外邦人说这话；因我是外邦人的使徒，所以敬重我的职分。 (罗马书 11:13)

保罗对自己的身份有一个认定，他说， “我是外邦人的使徒。 ” 他很清楚，自己从神那里领受的是使徒的呼召，并在外邦人，就是非犹太人中传扬神的福音。第二段经文在《启示录》里。

我知道你的行为、劳碌、忍耐，也知道你不能容忍恶人。你也曾试验那自称为使徒却不是使徒的，看出他们是假的来。 (启示录 2:2)

在初代教会，有很多人宣称自己是使徒，宣称他们写的东西非常重要，他们讲的东西非常重要，但是他们的教导却是错误的。所以，我要让大家知道，当时保罗宣布自己是使徒，并不是因为他想要一个头衔，他想要获得别人的尊重，而是他要保证教会的纯正性。只有使徒的教训才是真正从神那里得到的启示。这些假使徒，这些宣称自己是使徒却不是使徒的，如果教会听了他们的教训是非常危险的，是会偏离真道的。所以当大家读到保罗的书信，他宣告自己是外邦人的使徒，不要把它解读为说：你看这个保罗，出口也挺狂妄的，即便你是使徒，你有必要这样炫耀吗？这不是炫耀，这是一个生死之战，他在保证教会的纯正性。在写信给罗马教会的时候，他宣布：我是耶稣基督的仆人。但是他说：奉召为使徒。在英文版本中，是“a servant of Christ Jesus, called to bean apostle” ，called 是什么意思？我不是自封的，我是被呼召的，我是被设立的，我是被拣选的。接下来，他说“set apart for the gospel of God”(Romans 1:1 ESV)，中文翻译是“特派传神的福音”，懂英文的，你可以去查一下set apart 是什么意思。“特派”，另外一个意思就是分离出来，分别出来，分别为圣。就是神把保罗特别挑出来，特别拣选出来，分别出来，为要传神的福音。我让大家看的就是这个词，“set apart”。接下来，看《加拉太书》的一段经文来解释这个“set apart”。

你们听见我从前在犹太教中所行的事，怎样极力逼迫残害神的教会。我又在犹太教中，比我本国许多同岁的人更有长进，为我祖宗的遗传更加热心。然而，那把我从母腹里分别出来、又施恩召我的神，既然乐意将他儿子启示在我心里，叫我把他传在外邦人中，我就没有与属血气的人商量。 (加拉太书 1:13-16)

保罗是什么时候被拣选，什么时候被 set apart，什么时候呼召临到他？是神考察他的作为，经过这一段时间的观察，然后觉得保罗这个人如果作使徒挺不错的，是吗？这是这段经文所讲的吗？弟兄姊妹来默想一下，在妈妈的肚子里的时候，这个孩子就知道他将来是使徒保罗；大家可以想象一下，保罗的妈妈，如果她预先可以知道这个，在保罗还在肚子里，就摸着肚子说： “使徒保罗啊，使徒保罗。 ”我让大家发挥一下想象力。

我们要思想，我们每个人的命定是从什么时候开始的。因为有的人看着自己的过去说，我这么失败，能为主做什么呢？你在母腹当中的时候，神已经拣选你了。神不是根据你后来的表现说要不要拯救你，神也不是考察你的过去来决定要不要使用你。我们再看《耶利米书》中的一句经文：

我未将你造在腹中，我已晓得你；你未出母胎，我已分别你为圣；我已派你作列国的先知。 (耶利米书 1:5)

你看，一样的概念， set apart，分别为圣。我告诉大家，不仅是保罗被 set apart，不仅是耶利米被 set apart，大家看到施洗约翰在母腹当中也被圣灵充满。在耶稣降生之前有很多预言，还有天使向玛利亚显现。耶稣的例子也是，还没有出生，甚至还没有成形，还没有成为胎儿，就被 set apart 。还有参孙的例子。除了圣经的例子，还有和忠的例子、还有张静姐、还有芳芳，还有 …我们每一个人都是被 set apart。神不是让我们先生下来，然后观察一下从幼儿园到大学你的成绩怎么样，你的工作经历怎么样，我要来看一看你是否能够胜任。弟兄姊妹们，千万不要讲我不能胜任，没有不能胜任的，神如果给你什么样的呼召，祂就会给你什么样的能力。神要让你做什么，祂不会让你有缺乏的。很多弟兄姊妹的问题，不在于没有能力，而在于你根本还搞不清楚神要你做什么。你要去寻求神，神啊，你对我生命的呼召是什么。对于保罗来讲，他是外邦人的使徒，他要为主的名受很多苦。对于我来讲，是什么？神啊，求你向我显明。不然我日子过了一天又一天，别人看我日子也过的不错，我自己却过的很苦闷。你知道为什么苦闷吗？因为你不知道神对你的呼召是什么，你不知道神将你 set apart 要做什么。神不可能让你降生下来，然后神在天上想： “哎呀，又一个人生下来了，我接下来要做什么呢？ ”我告诉你，神如果没有想好要让你做什么，你不会出生的，你不会来到这个世界的。神对你有一个美好的旨意，在每个人的身上。

在我们的教会里面，弟兄姊妹如果出来服侍，我们看的不是他的能力怎么样，我们第一个要考察的是他的呼召，他有没有这个呼召。当一个姊妹来找我，她说她想去马尼拉读书。当时她在厦门的一个美国企业，她的工作前程不错，所以没有必要辞掉工作去读神学。你出来我们只能给你一个大学生刚毕业很微薄的工资，除非上帝呼召你，否则我们不要干这个事。所以有一年的时间她持续来找我们，我们给她泼冷水，如果是神的呼召，是不会灭掉的。所以弟兄姊妹们，你们放心，只要你寻求神，神会将祂的旨意很清楚地告诉你。神不会说，我故意不让你知道，你猜，你猜是什么。我们盼望神的灵在很多弟兄姊妹当中运行，将一批人呼召出来。呼召出来不是说，都辞职了，而是说，都明白神的心意，明白神要我做什么。我们看的是神在这个人生命当中的工作。如果一个人没有感动，你不要说“没有感动！你再说一遍！ ”有感动的人呢，早就感动得稀里哗啦了，不需要好像你硬要说服他什么，不需要。有时候当我们在分享一些经文的时候，当我们唱一些诗歌的时候，有些弟兄姊妹已经感动得不行了，为什么？因为神在对他说话。我们观察的不是这个人多么有才干、能力、恩赐，我们观察的是这个人身上是否有神的工作，神的时间。每个人都有呼召，有些人时间到了，有些人时间还没到。所以时间到的人，你可以看得到他对领受神的话语的那种心态，祷告的时候的那种感动，这些都说明神在他身上工作的时间到了。我们需要这些为神做工的人是内在发出力量，不是说照本宣科就开始卖弄他所学的一些东西，这不是由内在发出的一种力量。我们跟别人讲神的话语不是在背诵，而是这些话语激发出你里面的力量。一个被神呼召的人，他有内在发出的力量。我们要观察的是这个东西。有了这个东西，来服侍神他就非常有力量；没有这个东西，来服侍神他就很辛苦。所以我被神呼召，被神使用，我感到在我身上的一个使命就是，我要发出声音。以至于有些弟兄姊妹听到这些声音，他知道是神在呼召我，不是牧师在呼召我，不能再犹豫，不能再等待。也就是说，我们所做的工是被神使用，不是我能做什么，我是卑微的器皿。但是有没有可能神借着我要呼召一些弟兄姊妹，让他们不再停留在自己的安乐窝里面，让他们看到神对他的生命有个美好的旨意。教会如果是一群没有被神呼召的人去服侍、做事，这是一个悲剧。因为这些人不是用心做，而是用脑子做。他可以做得很好，看起来很好，但那是场面上的，真正神的工是做在一个人的灵魂之上。我们的讲道、祷告不是对人的脑袋说话，而是在对人的灵魂说话。被呼召的人，都是深深被神抓住的人。所以在祷告会里面，在很多场合里面，我们祷告的时候，我几乎可以看出哪些人是呼召出来的，因为这些人跟别人的反应是很不一样的。所以我告诉大家，有些人总是讲负面的例子，谁谁谁又软弱，谁谁谁又跌倒，不要再去看这些负面的东西。总有人软弱，总有人跌倒。但是我告诉你，总有人是被神呼召的，总有人是充满来自天上能力的！你要看这些人，你要看他们怎么生活，你要看他们怎么服侍神，你要看他们怎么跟随神，你才会得到鼓励。我们的眼目不能总是被那些负面的东西充满，因为这些人可能还不明白神，可能他的时间还没到。
