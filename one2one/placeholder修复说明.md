# Placeholder文字修复

## 📋 问题
- 文本框的placeholder显示为"请填写经文内容（当代译本）..."、"请填写经文内容（标准译本）..."等
- 应该统一显示"请填写经文内容（和合本）..."

## ✅ 修复内容

### 修改位置
`generate_course1_standard.py` - 两处placeholder

### 修改前
```python
placeholder="请填写经文内容（{verse['version']}）..."
placeholder="请填写经文内容（{section['verse_version']}）..."
```

### 修改后
```python
placeholder="请填写经文内容（和合本）..."
```

## 🎯 原因说明
- 虽然课程教材中使用了不同译本（标准译本、当代译本、新译本等）
- 但**批改标准始终是和合本**
- 为避免学员混淆，placeholder应该明确告知填写和合本经文

## ✨ 效果
现在所有15个经文填空框都统一显示：
```
请填写经文内容（和合本）...
```

这与批改逻辑一致，避免学员误以为要填写其他版本。
