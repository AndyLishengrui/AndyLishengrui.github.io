# 一对一门徒训练系统 (One-to-One Discipleship)

## 📖 项目简介
基于《一对一门徒训练》教材的交互式学习系统，提供经文填空、提示功能和答案检查等功能，帮助门徒建立扎实的属灵基础。

## 🎯 功能特色

### 📚 **交互式学习**
- **经文填空** - 核心经文的关键词填空练习
- **智能提示** - 提供字母提示帮助记忆
- **答案检查** - 即时验证填空答案
- **进度跟踪** - 记录学习进度和成绩

### 📱 **响应式设计**
- **移动优先** - 完美支持手机学习
- **多设备适配** - 平板、桌面全覆盖
- **现代界面** - 简洁美观的用户体验

### 🏗️ **模块化结构**
- **章节划分** - 按教材章节组织内容
- **独立页面** - 每章独立的学习页面
- **数据分离** - JSON格式存储课程数据

## 📁 目录结构

```
one2one/
├── data/              # 课程数据文件
│   ├── config.json    # 配置信息
│   └── chapter_*.json # 各章节数据
├── one2one_C1.html    # 第1章页面
├── one2one_C2.html    # 第2章页面
├── ...               # 其他章节
├── index.html        # 主页导航
└── README.md         # 项目说明
```

## 🔧 技术特性

### 前端技术
- **HTML5 + CSS3** - 现代Web标准
- **JavaScript ES6+** - 交互功能实现
- **响应式布局** - CSS Grid + Flexbox
- **本地存储** - 进度和成绩保存

### 数据格式
- **JSON数据** - 结构化课程内容
- **经文管理** - 标准化经文引用
- **答案系统** - 智能答案匹配

## 🚀 使用指南

1. **开始学习** - 从 `index.html` 进入
2. **选择章节** - 点击相应章节链接
3. **填空练习** - 在空白处填入答案
4. **获取提示** - 点击提示按钮获得帮助
5. **检查答案** - 验证填空结果
6. **跟踪进度** - 查看学习统计

## 📊 学习数据

系统将记录以下学习数据：
- ✅ 章节完成情况
- 📈 答题准确率
- ⏱️ 学习时间统计
- 🎯 重点经文掌握度
