## 🎯 学习方式

### 经文填空步骤

1. **阅读经文引用**（如"马太福音 5:21,22"）
2. **尝试填写**：凭记忆写出完整经文
3. **使用提示**（可选）：
   - 💡 渐进提示：逐步显示20%文字
   - 📝 查看答案：查看完整答案（和合本+其他译本）
4. **检查答案**：系统自动比对，70%相似度即可通过
5. **提交成绩**：保存学习进度

### 评分标准

- ✅ **完全正确**（绿色）：相似度≥90%
- ⚠️ **基本正确**（橙色）：70%≤相似度<90%
- ❌ **需要改进**（红色）：相似度<70%

## 💡 使用技巧

### ✨ 渐进提示功能

点击"💡 渐进提示"按钮，每次显示20%的答案：

- 第1次：显示前20%
- 第2次：显示前40%
- 第3次：显示前60%
- 第4次：显示前80%
- 第5次：显示完整答案

### 🔄 重复练习

1. 点击"🔄 清空答案"按钮
2. 答案框恢复空白状态
3. 可以重新填写练习

### 💾 自动保存

- 系统使用浏览器LocalStorage自动保存进度
- 下次访问同一页面时，答案会自动恢复
- 切换浏览器或设备，进度不会同步

## 🆘 常见问题

### Q: 页面显示"无法加载答案"？

**A:** 检查HTTP服务器是否正在运行：

```bash
lsof -i :8765  # 查看8765端口是否被占用
```

如果没有，重新启动服务器：

```bash
cd /Users/andyshengruilee/Documents/website/web2Lord/one2one
python3 -m http.server 8765
```

### Q: 经文填空不起作用？

**A:** 检查浏览器控制台是否有错误：

1. 按F12打开开发者工具
2. 切换到"Console"标签
3. 查看是否有红色错误信息
4. 刷新页面重试

### Q: 想重置所有学习进度？

**A:** 清除浏览器LocalStorage：

1. 按F12打开开发者工具
2. 切换到"Application"或"存储"标签
3. 找到"Local Storage"
4. 删除所有"one2one_"开头的条目

### Q: 手机上显示不正常？

**A:** 确保使用现代浏览器（Chrome、Safari）并刷新页面
